# Project Design
## Pages
- Calendar
    - Calendar view for the month/week where players can select days which pulls up the day view
    - Maybe only show the days the DM can run a session for simplicity
- Calendar Day
    - Page that contains all information about the given day.
    - What players are available that day, who can bring food/carpool
- Campaign History
    - List of all previous sessions
    - Section for DM campaign notes/timelines/routing
- Session Notes
    - List of notes left by the DM
